# ReinMorph3D Website Package

Upload the contents of this folder to GitHub Pages.

- `index.html`: homepage
- `assets/site.css`: page style
- `assets/site.js`: scroll reveal interaction
- `assets/main.pdf`: paper link target
- `assets/pipeline.pdf`: pipeline figure
- `assets/*.svg`: analysis figures

When you have videos, put the MP4 files into `assets/` and replace the video placeholder blocks in `index.html` with:

```html
<video autoplay playsinline loop muted controls src="assets/your-video.mp4"></video>
```
